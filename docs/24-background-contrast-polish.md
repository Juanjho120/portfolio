# Phase 8D - Background Contrast Polish

## Objective
Improve text readability before starting SEO work by reducing the brightness of the animated PCB background and strengthening contrast for the hero and admin dashboard headings.

## Changes
- Reduced global teal/cyan background glow intensity.
- Reduced PCB grid and trace opacity.
- Replaced the bright readability veil with a darker, subtler overlay.
- Made the hero title and description explicitly light-colored with controlled text shadow.
- Restyled the hero contact CTA as a dark glass button so it remains visible.
- Wrapped the admin analytics page heading block in a dark glass panel and improved the contrast for the title, eyebrow, description and latest matching event badge.

## Notes
This phase does not change PCB routing geometry, analytics event collection, Supabase persistence or admin access middleware. It is only a visual contrast pass before Phase 9 SEO and polish work.
