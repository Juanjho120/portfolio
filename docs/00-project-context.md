# Portfolio Context

This repository contains the main developer portfolio for Juan Tzun.

The portfolio will be deployed at:

- https://juantzun.dev

## Main Goals

- Present professional software projects as product-like case studies.
- Link each project to its live demo.
- Show screenshots, architecture, stack, CI/CD status, and GitHub metrics.
- Track visits and project card clicks.
- Serve as the central entry point for all portfolio projects.

## Primary Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel
- Supabase
- GitHub API

## Current Setup

The project was created with `create-next-app`.

Selected options:

- TypeScript: Yes
- ESLint: Yes
- React Compiler: No
- Tailwind CSS: Yes
- `src/` directory: No
- App Router: Yes
- Import alias: `@/*`
- AGENTS.md: Yes

## Portfolio Domain

Main domain:

- https://juantzun.dev

Project subdomains will be configured under the same root domain.
